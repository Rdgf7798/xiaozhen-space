## 部署步骤

### 第一部分：部署后端服务器

后端服务器负责存储视频文件和视频数据。

#### 方式一：Railway（推荐）

1. **准备后端代码**
   - 将 `/workspace/server` 文件夹上传到 GitHub 仓库
   - 仓库名称建议：`xiaozhen-space-server`

2. **创建 Railway 项目**
   - 访问 [railway.app](https://railway.app)
   - 使用 GitHub 账号登录
   - 点击 "New Project" → "Deploy from GitHub repo"
   - 选择你刚才创建的仓库
   - Railway 会自动检测 Node.js 并部署

3. **获取后端 URL**
   - 部署完成后，在项目设置中找到域名
   - 类似：`https://xiaozhen-server.up.railway.app`

4. **配置环境变量（可选）**
   - 如果需要修改端口，在 Railway 环境变量中添加：`PORT=3001`

#### 方式二：Render

1. **创建 Render 账号**
   - 访问 [render.com](https://render.com)
   - 使用 GitHub 账号登录

2. **创建 Web Service**
   - 点击 "New" → "Web Service"
   - 连接你的后端 GitHub 仓库
   - 设置：
     - **Root Directory**: `server`
     - **Build Command**: `npm install`
     - **Start Command**: `npm start`
   - 选择免费套餐
   - 点击 "Create Web Service"

3. **获取后端 URL**
   - 类似：`https://xiaozhen-space.onrender.com`

---

### 第二部分：部署前端

前端是一个纯静态网站，可以部署到 Vercel、Netlify 等平台。

#### 方式一：Vercel（推荐）

1. **准备前端代码**
   - 将 `/workspace` 文件夹上传到 GitHub 仓库（不包括 `server` 文件夹）
   - 仓库名称建议：`xiaozhen-space`

2. **配置环境变量**
   - 在 GitHub 仓库的 "Settings" → "Secrets and variables" → "Actions" 中添加：
     - **Name**: `VITE_API_URL`
     - **Value**: 你的后端 URL（例如：`https://xiaozhen-server.up.railway.app`）

3. **创建 Vercel 项目**
   - 访问 [vercel.com](https://vercel.com)
   - 使用 GitHub 账号登录
   - 点击 "Add New Project"
   - 导入你的前端仓库
   - 在 "Environment Variables" 中添加：
     - **Name**: `VITE_API_URL`
     - **Value**: 你的后端 URL
   - 点击 "Deploy"

4. **获取前端 URL**
   - 部署完成后，类似：`https://xiaozhen-space.vercel.app`

#### 方式二：Netlify

1. **构建前端**
   在本地项目目录执行：
   ```bash
   cd /workspace
   pnpm build
   ```

2. **部署到 Netlify**
   - 访问 [netlify.com](https://netlify.com)
   - 注册/登录账号
   - 点击 "Add new site" → "Deploy manually"
   - 将 `/workspace/dist` 文件夹拖入
   - Netlify 会提供类似 `https://random-name.netlify.app` 的 URL

3. **配置后端地址**
   - 在 Site Settings → Environment variables 中添加：
     - `VITE_API_URL` = 你的后端 URL

## 常见问题

### Q1: 视频上传失败？
检查：
1. 后端服务器是否正常运行
2. CORS 配置是否正确
3. 网络连接是否正常

### Q2: 视频播放不了？
检查：
1. 视频格式是否支持（推荐 MP4）
2. 视频文件是否上传成功
3. 后端服务器是否在运行

### Q3: 如何修改密码？
- 普通密码：编辑 `src/stores/authStore.ts`，修改 `DEFAULT_PASSWORD`
- 管理员密码：编辑 `src/stores/authStore.ts`，修改 `ADMIN_PASSWORD`
