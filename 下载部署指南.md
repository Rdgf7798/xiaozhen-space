# 📥 下载和部署指南

## 第一步：下载代码

访问工作空间，下载以下文件夹：
- `/workspace/xiaozhen-space.zip` - 完整代码

## 第二步：推送到 GitHub

### 创建后端仓库
1. 打开 https://github.com/new
2. Repository name: `xiaozhen-server`
3. 点 Create repository

### 创建前端仓库  
1. 打开 https://github.com/new
2. Repository name: `xiaozhen-space`
3. 点 Create repository

### 推送代码（需要安装 Git）

```bash
# 配置 Git
git config --global user.name "Rdgf7798"
git config --global user.email "你的邮箱"

# 解压代码
unzip xiaozhen-space.zip
cd xiaozhen-space

# 推送前端
cd .. 
# 在这里上传 xiaozhen-space 文件夹内容到 xiaozhen-space 仓库

# 推送后端
cd server
# 在这里上传 server 文件夹内容到 xiaozhen-server 仓库
```

## 第三步：部署

### 后端 - Railway
1. 打开 https://railway.app
2. Sign up → Deploy from GitHub → 选择 xiaozhen-server
3. 复制显示的 URL

### 前端 - Netlify
1. 打开 https://netlify.com
2. Sign up → Import from GitHub → 选择 xiaozhen-space
3. Build command: `pnpm install && pnpm build`
4. Publish directory: `dist`
5. 添加环境变量：
   - Key: `VITE_API_URL`
   - Value: Railway 的 URL
