# 🚀 小桢空间 - 一键部署指南

**预计完成时间：15分钟**

---

## 📋 部署概览

你需要部署两部分：
1. **后端服务器**（Railway）- 存储视频
2. **前端网站**（Netlify）- 用户界面

---

## 🎯 第一步：部署后端（Railway）

### 1.1 创建 GitHub 仓库

1. 打开 **[github.com/new](https://github.com/new)**

2. 填写信息：
   - **Repository name**: `xiaozhen-server`
   - **Description**: 小桢空间后端服务器
   - **Private/Public**: Public（公开）
   - **不要勾选** "Add a README file"

3. 点击 **"Create repository"**

4. 在新页面找到 **"…or push an existing repository from the command line"** 部分，复制命令

5. 在你的电脑上打开终端，运行复制的命令（大约是）：
   ```bash
   git remote add origin https://github.com/你的用户名/xiaozhen-server.git
   git branch -M main
   git push -u origin main
   ```

### 1.2 部署到 Railway

1. 打开 **[railway.app](https://railway.app)**

2. 点击 **"Sign Up"**（注册）
   - 可以用 GitHub 账号登录（最简单）

3. 登录后点击 **"New Project"**

4. 选择 **"Deploy from GitHub repo"**

5. 在列表中找到并点击 `xiaozhen-server` 仓库

6. Railway 会自动检测 Node.js 并开始部署！

7. 等待 1-2 分钟，直到显示绿色的 "Deployed"

8. **记录后端地址**：
   - 在项目页面找到 **"Domains"** 部分
   - 你会看到一个 URL，类似：`https://xiaozhen-server.up.railway.app`
   - **把这个地址复制下来，后面会用到！**

---

## 🎯 第二步：部署前端（Netlify）

### 2.1 创建 GitHub 仓库

1. 打开 **[github.com/new](https://github.com/new)**

2. 填写信息：
   - **Repository name**: `xiaozhen-space`
   - **Description**: 小桢空间视频分享平台
   - **Private/Public**: Public（公开）
   - **不要勾选** "Add a README file"

3. 点击 **"Create repository"**

4. 在新页面找到 **"…or push an existing repository from the command line"** 部分，复制命令

5. 在你的电脑上打开终端，运行（把 `你的用户名` 改成你的 GitHub 用户名）：
   ```bash
   git remote add origin https://github.com/你的用户名/xiaozhen-space.git
   git branch -M main
   git push -u origin main
   ```

### 2.2 部署到 Netlify

1. 打开 **[netlify.com](https://netlify.com)**

2. 点击 **"Sign up"**（注册）
   - 可以用 GitHub 账号登录

3. 登录后点击 **"Add new site"** → **"Import an existing project"**

4. 点击 **"Authorize Netlify"**（授权 GitHub）

5. 在列表中找到并点击 `xiaozhen-space` 仓库

6. **重要：配置构建设置**
   
   在 "Build settings" 中填写：
   - **Build command**: `pnpm install && pnpm build`
   - **Publish directory**: `dist`

7. **添加环境变量**：
   - 点击 **"Add environment variable"**
   - **Key**: `VITE_API_URL`
   - **Value**: 你在 Railway 得到的后端地址
     - 例如：`https://xiaozhen-server.up.railway.app`
   - 点击 **"Add environment variable"** 保存

8. 点击 **"Deploy site"**

9. 等待 2-3 分钟，直到显示 "Published"

10. **记录前端地址**：
    - 在页面顶部会显示你的网站 URL
    - 类似：`https://random-name.netlify.app`
    - 或者你可以点击 **"Site settings"** → **"Change site name"** 改成你喜欢的名字
    - 例如改成 `xiaozhen-space`，就会变成：`https://xiaozhen-space.netlify.app`

## 🔧 常见问题

### Q1: Railway 部署失败？
- 检查 GitHub 仓库是否包含 `server.js` 和 `package.json`
- 检查 Railway 日志中的错误信息
- 确保仓库是公开的

### Q2: 前端显示"无法连接服务器"？
- 检查 Vercel/Netlify 的环境变量 `VITE_API_URL` 是否正确设置
- 确保后端 Railway 已经部署成功

### Q3: 视频上传失败？
- 检查后端服务器是否正常运行
- 检查 Railway 是否有足够的存储空间
- 视频大小限制：500MB

### Q4: 如何修改密码？
- **普通密码**：编辑 `src/stores/authStore.ts`，找到 `DEFAULT_PASSWORD`
- **管理员密码**：编辑 `src/stores/authStore.ts`，找到 `ADMIN_PASSWORD`
- 修改后需要重新部署前端

### Q5: 如何删除已上传的视频？
- 需要用管理员密码 `20100719zzq` 登录
- 进入"上传视频"页面
- 找到要删除的视频，点击删除按钮
