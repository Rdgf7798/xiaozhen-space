# 🚀 小桢空间 - 极简部署指南（手把手）

## ⚡ 你只需要做这5步

---

## 第一步：创建后端仓库（上GitHub）

1. 打开这个链接：[github.com/new](https://github.com/new)

2. 填写：
   - **Repository name**: `xiaozhen-server`
   - 其他都不要改
   - 点 **Create repository**

3. **复制页面上的命令**，类似：
   ```bash
   git remote add origin https://github.com/你的用户名/xiaozhen-server.git
   git push -u origin main
   ```

---

## 第二步：创建前端仓库（上GitHub）

1. 再打开这个链接：[github.com/new](https://github.com/new)

2. 填写：
   - **Repository name**: `xiaozhen-space`
   - 其他都不要改
   - 点 **Create repository**

3. **复制页面上的命令**，类似：
   ```bash
   git remote add origin https://github.com/你的用户名/xiaozhen-space.git
   git push -u origin main
   ```

---

## 第三步：推送代码

在**终端**（Mac叫终端，Windows叫PowerShell）运行：

### 推送后端：
```bash
# 1. 进入后端目录
cd server

# 2. 初始化git（如果还没初始化）
git init

# 3. 添加文件
git add .

# 4. 提交
git commit -m "后端服务器"

# 5. 推送（把下面的URL换成你在第一步复制的）
git remote add origin https://github.com/你的用户名/xiaozhen-server.git
git push -u origin main
```

### 推送前端：
```bash
# 1. 进入前端目录（回到项目根目录）
cd ..

# 2. 添加文件
git add .

# 3. 提交
git commit -m "前端网站"

# 4. 推送（把下面的URL换成你在第二步复制的）
git remote add origin https://github.com/你的用户名/xiaozhen-space.git
git push -u origin main
```

---

## 第四步：部署后端（Railway）

1. 打开：[railway.app](https://railway.app)
2. 点 **Sign Up**，用 GitHub 登录
3. 点 **New Project** → **Deploy from GitHub repo**
4. 选择 `xiaozhen-server` 仓库
5. 等待部署完成（1-2分钟）
6. **复制显示的 URL**，类似：`https://xiaozhen-server.up.railway.app`

---

## 第五步：部署前端（Netlify）

1. 打开：[netlify.com](https://netlify.com)
2. 点 **Sign Up**，用 GitHub 登录
3. 点 **Add new site** → **Import an existing project**
4. 选择 `xiaozhen-space` 仓库
5. **Build settings** 填写：
   - Build command: `pnpm install && pnpm build`
   - Publish directory: `dist`
6. 点 **Add environment variable**：
   - Key: `VITE_API_URL`
   - Value: **粘贴你在第四步复制的URL**
7. 点 **Deploy site**
8. 等待部署完成（2-3分钟）
